Durante este curso aprendí que la tecnología puede ser una herramienta muy útil para fortalecer la participación ciudadana y la organización comunitaria. También comprendí la importancia de usar bases de datos para organizar la información y facilitar el seguimiento de las iniciativas. Aprendí cómo Git y GitHub ayudan a controlar versiones y compartir proyectos. Conocí herramientas como n8n y Telegram para automatizar procesos y mejorar la comunicación. Además, entendí la importancia de proteger los datos personales y manejar la información de forma responsable. Todos estos conocimientos permiten crear soluciones tecnológicas que beneficien a las comunidades y hagan más eficiente la participación de los ciudadanos.
